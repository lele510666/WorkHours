# Official Page for WorkHours App

WorkHours is a productivity app designed to help users track daily work hours, manage projects, and calculate income based on recorded time.

It is suitable for: 
- Full-time workers
- Part-time workers
-	Freelancers
- Anyone who wants a clear and structured overview of their working time


# Privacy Policy

WorkHours respects your privacy.

This app does not collect, store, or transmit any personal data to external servers.


# Data Usage

All data created or used by WorkHours is processed locally on your device.

-	Work hour records, project data, and app settings are stored on the device
-	If iCloud Sync is enabled, data is synchronized securely through Apple’s iCloud service only
-	No data is uploaded to any third-party server
-	No analytics, advertising, or tracking services are used
-	No user account or login is required to use the app


# Permissions Explanation

WorkHours may request access to the following system permissions only to provide core functionality.

## Location (When In Use / Always)

Location access is used to:

-	Search for and select work locations
-	Support automatic clock-in and clock-out features
-	Detect location changes related to your chosen work location

Location data is processed locally and is never shared or stored on external servers.

## Other Data
-	WorkHours does not access contacts, photos, files, or personal identifiers
-	All permissions are optional and can be managed or revoked at any time in your device’s system settings
